# analise-vendas-python
Análise de dados de vendas com Python e pandas

Projeto simples de análise de dados utilizando Python e pandas.

# O que o projeto faz:
- Lê dados de vendas de um arquivo CSV
- Classifica vendas em categorias (Baixo, Médio, Alto)
- Calcula faturamento por categoria
- Calcula quantidade de vendas
- Gera relatório com ticket médio

# Tecnologias usadas:
- Python
- Pandas

# Arquivos gerados:
- faturamento_categoria.csv
- quantidade_vendas.csv
- relatorio.csv
